# Zigbee Lock Samsung SDS (SHM)

삼성 SDS Zigbee 도어록용 SmartThings Edge 드라이버입니다.
SmartThings 홈 모니터링(SHM)과 연동하여 외출방범/재택안심 모드를 지원합니다.

## 지원 기기

* 삼성 SDS SHP-DP960SG (DADT302 Zigbee 모듈) — 실기기 테스트 완료
* 기타 삼성 SDS Zigbee 도어록 (동일 모듈 사용 제품)

> Zigbee 모듈(DADT302) 장착 기기 전용입니다. Wi-Fi 모듈 장착 기기는 직방 앱을 통해 별도로 연동하며, 본 드라이버와는 관계없습니다.

## 주요 기능

* 지문 / 비밀번호 / 앱 원격 / 내부에서 열림 구분 기록
* 사용자별 재실 센서 자동 생성 (지문 슬롯 번호 + 이름 등록)
* 자동잠금 감지
* 외출방범 / 재택안심 모드 SmartThings 홈 모니터링 연동

### 홈 모니터링(SHM) 연동 동작

| 보안 모드 | 열림 수단 | 결과 |
|---------|----------|------|
| 안심(외출) | 지문 / 비밀번호 / 앱 | 보안 모드 해제 (경보 없음) |
| 안심(외출) | 내부에서 열림 | 경보 유지 (침입 감지) |
| 안심(외출) | 강제 개방 (모티스) | 침입 경보 발동 |
| 안심(실내) | 내부에서 열림 | 보안 모드 해제 (경보 없음) |
| 안심(실내) | 강제 개방 (모티스) | 침입 경보 발동 |
| 보안 해제 | 강제 개방 (모티스) | 기록만 남음 (경보 없음) |

도어록 본체의 외출방범/재택안심 버튼으로 SHM 모드가 자동 설정됩니다.

## 설치 방법

1. [채널 등록](https://bestow-regional.api.smartthings.com/invite/r3MyN87OZJ2p) 후 허브에 Enroll
2. SmartThings 앱 → 기기 → 도어록 → 드라이버 선택 → `Zigbee Lock Samsung SDS (SHM)` 설치

## 사용자 재실 센서 설정

설치 후 기기 설정(Preferences)에서 슬롯 번호와 이름을 입력하면 사용자별 재실 센서가 자동 생성됩니다.
슬롯 번호 확인 방법: 드라이버 설치 후 지문으로 열면 앱 기록에 `잠금 해제됨, 슬롯 N, 지문` 으로 표시됩니다.

## 홈 모니터링 설정 주의사항

SHM 연동이 올바르게 동작하려면 아래 설정이 필요합니다.

**반드시 선택**
* 보안 모드 연동 기기 → Samsung Door Lock
* 안심(외출/실내) → 잠금 선택 → Samsung Door Lock

**반드시 선택 안 함**
* 재실 감지 센서 → 차일드 기기(사용자 재실 센서) 선택 시 오경보 발생
* 동작 / 접촉 / 소리 센서 → 도어록 관련 센서 선택 금지

## 직방 앱 보안 모드와의 차이

직방 앱은 도어록 펌웨어와 직접 통신하여 하드웨어 레벨에서 보안 모드를 제어합니다.
본 드라이버의 SHM 연동은 Zigbee로 수신된 이벤트를 SmartThings가 소프트웨어적으로 판단하는 방식으로, 아래와 같은 차이가 있습니다.

| 기능 | 직방 앱 | 본 드라이버 (SHM) |
|------|--------|----------------|
| 외출방범 + 인증 귀가 | 경보 없음 | 경보 없음 (소프트웨어 처리) |
| 외출방범 + 내부에서 열림 | 경보 | 경보 (소프트웨어 처리) |
| 재택안심 + 외부 침입 시도 차단 | 하드웨어 차단 + 경보 | 하드웨어 차단은 되나 SmartThings 경보 없음 |
| 비밀번호 사용자 구분 | 지원 | 미지원 (user_id=0 고정) |
| 보안 모드 원격 설정 | 지원 | 미지원 (Zigbee 한계) |

## 한계 (실기기 확인)

* 비밀번호 사용자 구분 불가 (user_id=0 고정)
* 외출방범 / 재택안심 모드 원격 설정 불가 (Zigbee 미지원)
* 재택안심 모드 중 외부에서 지문/비밀번호 시도 시 도어록 하드웨어가 차단하며 앱으로도 열기 불가
* 재택안심 중 앱 열기 시도 시 즉시 잠김 상태로 복원되나 앱 UI 타임아웃으로 인해 짧은 빙글빙글 후 오류 메시지 표시됨 (SmartThings 앱 레벨 한계로 드라이버로 완전 해결 불가)
* 보안 모드 원격 설정 시 도어록 본체 LED/음성 안내 없음 (소프트웨어 레벨만 변경됨)
* NFC (카드형 태그) — 동작 여부 미확인


## 주의사항 및 면책 조항

본 드라이버는 개인이 학습 및 편의를 위해 개발한 오픈소스 소프트웨어입니다.
드라이버 사용으로 인해 발생하는 기기 오작동, 보안 사고, 또는 하드웨어 손상에 대해 개발자는 어떠한 법적 책임도 지지 않습니다.
설치 및 사용에 대한 모든 책임은 사용자 본인에게 있으며, 중요 보안 시설에는 사용을 권장하지 않습니다.
삼성 SDS의 공식 드라이버가 아니며, 제조사의 업데이트에 따라 기능이 제한될 수 있습니다.
